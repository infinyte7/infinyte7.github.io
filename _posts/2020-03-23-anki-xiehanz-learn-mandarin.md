Learn, read, write and practice Mandarin by drawing strokes in Anki Desktop, AnkiDroid and AnkiMobile with audio of HSK1 to HSK9 characters.  

#### Download From<br>
<a href="https://github.com/infinyte7/Anki-xiehanzi"><button type="button" class="btn btn-dark">GitHub</button></a>
<a href="https://ankiweb.net/shared/info/119943820"><button type="button" class="btn btn-dark">AnkiWeb</button></a>

#### Features
- Night mode
- Change size of characters
- Change drawing stroke width
- HSK1 - HSK9 audio included in the decks
- View meaning by opening Pleco dictionary on phone.
- View character details using [https://hanzicraft.com/](hanzicraft)
- View mnemonics of characters using [http://rtega.be/chmn/](http://rtega.be)
- Practice simplified as well as traditional characters
- Show or hide Simplified, Traditional characters, Pinyin or Meaning
- Draw characters to learn Mandarin with Simplified, Traditional, Pinyin and Meaning
- Indicator at bottom for showing if character loaded or not `(green - loaded, red - not loaded or some error)`

#### Note:
Before importing Anki Deck please make backups with scheduling information.
